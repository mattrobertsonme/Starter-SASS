This is the sass template v1 by Matt Robertson. 

The sass template v1 acts serves the main css for the Themes Plus network.

In Koala set the Output Style to either expanded or nested.

Inside your text editor use the equals sign to quickly navigate between sections.

File Structure:

|- style.scss
   |-- variables
      |-- colors
      |-- flex
      |-- stucture 
      |-- typography
   |- mixins
